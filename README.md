
# PIZZON

The Pizzon Website Project is a modern online platform developed with HTML, CSS, TypeScript, Bootstrap, React, Git, and GitHub. It provides a seamless user experience across devices, offering dynamic content like menus and special offers. Integrated with a RESTful API for food, users can explore a wide range of dishes and ingredients. With the added cart feature, selecting items and checkout processes are effortless. Managed with Git and deployed on GitHub Pages, the project ensures efficient development and accessibility for diners.
